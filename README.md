## 项目简介

这个博客是我基于开源项目进行学习和搭建的个人网站。  
在搭建过程中，我参考了https://github.com/eonova/eonova.me  的整体框架设计与部分实现思路，在此对原作者表示感谢。

原项目本身是一个个人博客与作品集网站，包含博客展示、项目管理、评论互动等功能，技术栈也比较现代，对我理解个人博客项目的结构和开发流程帮助很大。:
